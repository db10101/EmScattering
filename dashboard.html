Overview

Select number of layers of your structure. Press "Submit Number of Layers." This will return forms equal to the number of layers plus 2 for ambient layers.

Next, enter in Epsilon, Mu, and Lengths for all layers and the two ambients. Then, select "Create Structure."

If successful, a dialog box will pop up letting you know and the tabs "Scattering" and "Guided Modes" will appear.

If you create a structure with too many layers, refresh the page to restart the process.
